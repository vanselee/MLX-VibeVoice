# 音频停顿与角色切换节奏验收方案

**文档版本**: v1.0
**创建日期**: 2026-05-06
**测试目标**: 验证不同音色/不同角色切换时，导出 WAV 是否有自然停顿

---

## 1. 测试目的

验证 AudioPauseService 和 AudioExportService 的停顿注入逻辑：
- 不同角色间切换时停顿时长是否合理
- 手动停顿标签是否正确解析
- 中文标点是否触发停顿微调
- 停顿标签是否被模型读出（不应读出）
- 导出 WAV 是否可正常播放

---

## 2. 测试样例

### 样例 A：同角色连续段落

**场景**: 同一角色连续说两段话
**预期**: 短停顿（约 150-250ms）

```
角色：主持人
内容：
第一段：大家好，欢迎来到今天的节目。
第二段：今天我们聊一个很有意思的话题。
```

**验证点**:
- [ ] 两段之间有停顿，但不是硬拼接
- [ ] 停顿不过长，听起来自然连贯

---

### 样例 B：双角色对话

**场景**: 两个角色交替对话
**预期**: 中等停顿（约 300-500ms）

```
段落1：
角色：主持人
内容：李老师，您怎么看这个问题？

段落2：
角色：李老师
内容：我觉得这个问题要从几个方面来分析。

段落3：
角色：主持人
内容：能具体说说吗？
```

**验证点**:
- [ ] 角色切换处有明显停顿
- [ ] 听起来像自然对话节奏
- [ ] 不会感觉两个声音"撞在一起"

---

### 样例 C：旁白与角色切换

**场景**: 旁白叙述后接角色对话
**预期**: 稍长停顿（约 400-600ms）

```
段落1：
角色：旁白
内容：会议室里，大家都在等待发言。

段落2：
角色：张总
内容：好，那我们开始今天的讨论。

段落3：
角色：旁白
内容：张总的话音刚落，李经理就举起了手。
```

**验证点**:
- [ ] 旁白→角色有明显停顿
- [ ] 角色→旁白有明显停顿
- [ ] 区别于同角色连续段落的短停顿

---

### 样例 D：手动停顿标签

**场景**: 文案中包含手动停顿标签
**预期**: 标签被解析为停顿，不被模型读出

```
角色：主持人
内容：
大家好，欢迎来到今天的节目。[停顿短]
今天我们聊一个很有意思的话题。[停顿中]
这个话题就是——[停顿长]
如何在繁忙的工作中保持高效。
另外，[停顿800ms]我想请大家注意一个细节。
```

**验证点**:
- [ ] `[停顿短]` 解析为 ~200ms 停顿
- [ ] `[停顿中]` 解析为 ~400ms 停顿
- [ ] `[停顿长]` 解析为 ~800ms 停顿
- [ ] `[停顿800ms]` 解析为 800ms 停顿
- [ ] 标签文字没有被模型读出来（关键！）
- [ ] 超过 3000ms 的值被 clamp 到 3000ms

---

### 样例 E：中文标点停顿

**场景**: 段落结尾包含中文标点
**预期**: 标点触发停顿微调

```
角色：主持人
内容：
你真的准备好了吗？(问号 → +100ms)
太棒了！(感叹号 → +80ms)
其实……(省略号 → +150ms)
```

**验证点**:
- [ ] 问号结尾停顿比句号稍长
- [ ] 感叹号结尾停顿比句号稍长
- [ ] 省略号结尾停顿明显加长
- [ ] 中英文标点效果一致

## 3. 验收标准

| 标准 | 说明 |
|------|------|
| 自然切换 | 不同角色切换听起来不硬拼，有自然间隔 |
| 停顿合理 | 停顿不拖沓，不会让人感觉"等太久" |
| 标签静默 | 停顿标签 `[停顿xxx]` 不会被模型读出来 |
| WAV 可播放 | 导出文件可被系统播放器/其他播放器正常播放 |
| 无爆音/杂音 | 段落拼接处无爆音、无杂音、无异常跳变 |

## 6. 附录：停顿配置参考

来自 `AudioPauseService.swift` 的 PauseConfig：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| AutoPause.sameRole | 200ms | 同角色连续段落 |
| AutoPause.differentRole | 400ms | 不同角色切换 |
| AutoPause.narratorTransition | 500ms | 旁白↔角色切换 |
| Punctuation.questionMark | 100ms | 问号结尾加成 |
| Punctuation.exclamationMark | 80ms | 感叹号结尾加成 |
| Punctuation.ellipsis | 150ms | 省略号结尾加成 |
| Manual.short | 200ms | [停顿短] |
| Manual.medium | 400ms | [停顿中] |
| Manual.long | 800ms | [停顿长] |
| Max.autoPause | 3000ms | 自动停顿上限 |
| Max.manual | 3000ms | 手动停顿上限 |

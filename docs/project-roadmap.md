## 重要约束

### ❌ 不要做这些

- 不删除 `default.store`，里面有旧测试数据
- 不做旧数据迁移，除非用户明确要求
- 不要删除 Application Support 数据
- 不要引入其他模型，仅用 Qwen3 bf16
- 不要用 voice instruct，已验证不稳定
- 不要导出分段音频，用户只需要完整 WAV
- 不要引入 Phase 3，不要碰 AI 指导、角色设计、云推理等

### ✅ 一定要做这些

- 每次修改必须 git commit，中文提交信息
